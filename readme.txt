********************************************************
STEPS FOR RUNNING THE APPLICATION
********************************************************
1. Inside the SSDI project, you can find all the source code inside the 'Source code' folder. 
2. In the Mongo shell, execute all the commands present in mongoDB text file inside 'Source code' folder.
3. Unzip the '49erKart.zip' and open the file in VS code.
4. Open the terminal inside the project folder,
	Run 'npm init'.
	Then, run the command 'node app.js'
5. Open the browser and give the URL as 'http://localhost:8080/'.
